搭建个人图床

# 搭建个人图床

# 0 前言

为什么选择阿里云oss？

  1. 阿里云图床是大公司产品，比较稳定。

  2. 价格在能接受的范围内。9 元/年，存储空间为 40 G

# 1 准备图床

# 1.1购买阿里云oss

费用表：1gb 0.12元/月，对于我这种博客用户需求量不是特别大可以接受。

# 1.2 新建 Bucket

bucket：相当于一个存储图片的房子。下图就是配置这个房子，通过配置bucket我们可以传递图片到这个bucket里

​ （这里忘记截图了）

# 2 使用 PicGo 上传图片到图床

# 2.1下载装PicGo

PicGo官网 (opens new window)

Setup（箭头）指的是正式版，mac根据需求下载其他版本

# 2.2配置PicGo

  • accesskey和accesskeySecret可以从阿里云控制台获取。
  • 存储空间名是bucket的名字。
  • 存储区域到阿里云OSS控制台去找，例如下图中存储区域就是 oss-cn-beijing

  • 存储路径是存储图片的位置，要求以/结尾。
  • 自定义域名可以不填写。

# 3 配置typora自动更新将图片到图床

# 3. 1升级到最新版的typora

  • 帮助-检查升级

# 3.2 配置typora

点开文件-偏好设置，然后配置文件并点击下载或更新

# 3.3 配置PicGo

{
  "picBed": {
    "uploader": "aliyun",
    "aliyun": {
    "accessKeyId": "",
    "accessKeySecret": "",
    "bucket": "", // 存储空间名
    "area": "", // 存储区域代号
    "path": "img/", // 自定义存储路径
     "customUrl": "", // 自定义域名，注意要加 http://或者 https://
     "options": "" // 针对图片的一些后缀处理参数 PicGo 2.2.0+ PicGo-Core 1.4.0+
    }
  },
  "picgoPlugins": {}
}
